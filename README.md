### 医院管理系统小程序+uniapp+SpringBoot+vue
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL、VSCode、微信开发者工具/HBuilderx
技术栈：Java、MySQL、SpringBoot、Mybatis/MybatisPlus、Vue2、ElementUI、JavaScript、CSS、Uniapp

系统功能模块：
本系统包含管理员和用户2个角色。
1.管理员角色：个人中心，用户管理，医生管理，排班信息管理，医生信息管理，科室信息管理，科室预约管理，病历信息管理，体检报告管理，系统管理等。医生可以进行管理，排班，处理预约信息，体检信息，病历信息
2.用户角色：注册登录，进行预约
